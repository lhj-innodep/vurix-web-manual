고장 진단

고장 신고 전에 확인하세요.

일반적으로 발생하는 문제에 대해 가능한 원인과 해결 방법을 제시합니다.

증상 - 영상을 재생할 때 로딩이 오래 걸린다.

•장비를 삭제한 후 재할당하세요.

•증상이 계속되는 경우 웹 API 서버를 재시작하세요.

증상 - '네트워크 또는 서버 상태를 확인하세요.' 에러 메시지가 뜬다.

•웹 API 서버를 재시작하세요.

증상 - '입력 정보를 확인하세요.' 에러 메시지가 뜬다.

•VURIX 클라이언트에 등록된 아이디와 비밀번호를 확인하세요.

증상 - 이벤트 수신 설정을 했지만, 이벤트 수신이 되지 않는다.

•VURIX 클라이언트에 등록된 이벤트를 확인하세요.

•증상이 계속되는 경우 웹 API 서버를 재시작하세요.

증상 - 장비 목록에 원하는 장비가 없다.

•VURIX 클라이언트에서 장비 접근 제어 권한을 확인하세요.

증상 - PTZ 제어 권한이 없다.

•VURIX 클라이언트에서 PTZ 제어 권한을 확인하세요.

증상 - 연결 상태 불안정 안내 알림 창이 빈번하게 뜬다.

•장비 할당 상태를 확인하세요.

•증상이 계속되는 경우레코딩 서버를 재시작하세요.
